Guía de Uso del Sistema

1. Iniciar el sistema: Ejecuta Main.java. Se abrirá un menú interactivo en la consola.

2. Menú principal: Selecciona la opción deseada escribiendo el número correspondiente.

TareaFarmacia.java

Gestiona las tareas dentro del sistema de farmacia, incluyendo creación, procesamiento y finalización de tareas.

Node.java

Clase básica para nodos de estructuras de datos.

Pila.java

Implementa la estructura de datos de tipo pila para manejar tareas.

ListaTareas.java

Lista que administra las tareas pendientes en el sistema.

ColaPrioridad.java

Gestiona tareas según prioridad para un flujo más eficiente.

Empleado.java

Representa a los empleados del sistema de farmacia con sus atributos principales.

EmployeeNode.java

Nodo para estructuras que gestionan empleados.

EmployeeTree.java

Árbol binario que organiza empleados según sus características.

EmployeeManager.java

Gestiona empleados con árboles binarios y mapas, facilitando búsquedas y asignaciones.

Permite agregar empleados, asignar tareas de forma óptima, mostrar empleados disponibles por habilidades y distribuir cargas de trabajo mediante un algoritmo de divide y vencerás.

Main.java

Centraliza el sistema de gestión de farmacia mediante un menú en consola.

  • Agregar, procesar, completar y eliminar tareas.
  • Mostrar estadísticas de tiempo estimado.
  • Agregar empleados, buscarlos, mostrarlos por departamento o disponibilidad.
  • Asignar tareas óptimas a empleados.
