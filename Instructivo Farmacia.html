Guía de Uso del Sistema

  1. Iniciar el sistema: Ejecuta Main.java. Se abrirá un menú interactivo en la consola.
  2. Menú principal: Selecciona la opción deseada escribiendo el número correspondiente.

    Opciones comunes:

    • Agregar tarea: introduce ID, tipo, descripción, prioridad y tiempo estimado.
    • Procesar tarea: selecciona si es urgente, receta, prioritaria o general.
    • Completar/Eliminar tarea: ingresa el ID de la tarea a gestionar.
    • Mostrar estadísticas: revisa tareas pendientes y tiempo estimado total.
  3. Gestión de empleados:
    • Agregar empleado: introduce ID, nombre, departamento, puesto y habilidades.
    • Mostrar empleados por departamento: consulta la organización jerárquica.
    • Buscar por ID o habilidad: facilita localizar empleados específicos.
    • Asignar tareas: el sistema sugiere automáticamente el empleado más adecuado según carga y habilidades.
  4. Consejos prácticos:
    • Mantén actualizada la carga de trabajo de los empleados para una asignación óptima.
    • Completa las tareas a tiempo para mantener la lista organizada.
    • Usa búsquedas por tipo y descripción para encontrar tareas rápidamente.

TareaFarmacia.java

Representa una tarea dentro de la farmacia, con atributos como ID único, descripción, tipo (urgente, receta, inventario, administrativa o prioritaria), prioridad, fecha de creación, estado y tiempo estimado.

  • Constructor inicializa atributos y estado PENDIENTE.
  • Métodos getter y setter para acceder y modificar.
  • estaCompletada() verifica si está completada.
  • toString() muestra representación textual.

Node.java

Elemento básico para construir estructuras enlazadas (listas, pilas y colas). Contiene una TareaFarmacia y la referencia al siguiente nodo.

  • Constructor inicializa tarea y siguiente como nulo.
  • Getters y setters para tarea y siguiente nodo.

Pila.java

Representa una pila de tareas urgentes utilizando nodos enlazados.

  • Apilar nuevas tareas.
  • Procesar tarea más urgente.
  • Completar tarea por ID.
  • Consultar cima sin eliminar.
  • Verificar vacío y tamaño.
  • Mostrar tareas pendientes y completadas.

ListaTareas.java

Implementa lista enlazada para gestionar tareas generales.

  • Insertar al final.
  • Completar o eliminar por ID.
  • Buscar por ID, tipo o descripción.
  • Contar pendientes.
  • Mostrar pendientes y completadas por prioridad.
  • Calcular tiempo total pendiente (recursivo).
  • Obtener tareas ordenadas por prioridad y fecha.

ColaPrioridad.java

Cola de prioridad implementada con heap binario.

  • Insertar tareas.
  • Consultar mayor prioridad.
  • Procesar y completar por ID.
  • Mostrar todas ordenadas.
  • Comparación por estado → prioridad → fecha.

Empleado.java

Representa un trabajador de la farmacia con ID, nombre, departamento, puesto, habilidades y carga de trabajo.

  • Verificar disponibilidad.
  • Asignar carga.
  • Comprobar habilidades.
  • Representación textual del empleado.

EmployeeNode.java

Nodo de un árbol binario de empleados.

  • Contiene un Empleado.
  • Referencias a hijo izquierdo y derecho.
  • Base para estructuras jerárquicas y búsquedas.

EmployeeTree.java

Árbol binario que organiza empleados por criterios configurables (ID, nombre o carga de trabajo).

  • Insertar y buscar empleados.
  • Asignar tareas óptimas.
  • Consultar por departamento.
  • Mostrar estadísticas.

EmployeeManager.java

Gestiona empleados utilizando árboles binarios y mapas, optimizando búsquedas y asignaciones.

  • Agregar empleados.
  • Asignar tareas de forma óptima.
  • Mostrar empleados disponibles por habilidades.
  • Distribuir cargas de trabajo con algoritmo divide y vencerás.

Main.java

Clase central que ejecuta el sistema de gestión de farmacia a través de un menú en consola.

  • Agregar, procesar, completar y eliminar tareas.
  • Mostrar estadísticas de tiempo estimado.
  • Agregar y buscar empleados.
  • Mostrar empleados por departamento o disponibilidad.
  • Asignar tareas óptimas a empleados.
