Guía de Uso del Sistema

Explica cómo iniciar el sistema, usar el menú principal, gestionar empleados y consejos prácticos para optimizar el uso del software.

TareaFarmacia.java

Gestiona tareas en la farmacia con atributos como ID, tipo, descripción, prioridad, estado y tiempo estimado.

Node.java

Clase básica para estructuras enlazadas que almacena una tarea y referencia al siguiente nodo.

Pila.java

Maneja una pila de tareas urgentes, permitiendo apilar, procesar, completar y mostrar tareas.

ListaTareas.java

Lista enlazada para tareas generales con búsqueda, inserción, eliminación, estadísticas y ordenamiento.

ColaPrioridad.java

Cola de prioridad con heap binario para procesar primero las tareas más urgentes y antiguas.

Empleado.java

Representa a un trabajador con atributos como ID, nombre, puesto, departamento y habilidades.

EmployeeNode.java

Nodo para un árbol binario de empleados que contiene un empleado y referencias a hijos.

EmployeeTree.java

Árbol binario que organiza empleados por criterios como ID, nombre o carga de trabajo.

EmployeeManager.java

Gestiona empleados con árboles binarios y mapas para asignaciones óptimas y consultas rápidas.

Main.java

Clase principal que ejecuta el sistema con un menú en consola para manejar tareas y empleados.
