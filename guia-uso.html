Introducción al Sistema

El Sistema de Gestión de Farmacia es una aplicación Java diseñada para optimizar la administración de tareas y recursos humanos en una farmacia. Combina estructuras de datos eficientes con una interfaz intuitiva.

Características principales:
  • Gestión priorizada de tareas
  • Asignación inteligente de empleados
  • Seguimiento de estado en tiempo real
  • Estadísticas y reportes
  • Múltiples estructuras de datos optimizadas

Gestión de Empleados

Flujo del algoritmo de asignación

Algoritmo Divide y Vencerás

Para la distribución equitativa de carga de trabajo:

  1. Dividir - Separar empleados por departamento
  2. Conquistar - Asignar tareas dentro de cada grupo
  3. Combinar - Ajustar entre departamentos si es necesario
// Pseudocódigo del algoritmo
public void distribuirCarga() {
  if (empleados.size() <= 1) return;
  
  List mitad1 = dividir(empleados, 0, mitad);
  List mitad2 = dividir(empleados, mitad, empleados.size());
  
  distribuirCarga(mitad1);
  distribuirCarga(mitad2);
  
  combinar(mitad1, mitad2);
}

Resolución de Problemas

Problemas Comunes y Soluciones

Error: "Memoria insuficiente"
Solución: Aumentar la memoria asignada a la JVM usando -Xmx parameter
Error: "Tarea no encontrada"
Solución: Verificar que el ID de tarea existe y está correctamente escrito
Error: "Empleado no disponible"
Solución: Revisar carga de trabajo o agregar más empleados al sistema
Error: "Estructura de datos llena"
Solución: Completar tareas pendientes o aumentar límites en configuración
