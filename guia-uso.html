Introducción al Sistema

El Sistema de Gestión de Farmacia es una aplicación Java diseñada para optimizar la administración de tareas y recursos humanos en una farmacia. Combina estructuras de datos eficientes con una interfaz intuitiva.

Características principales:
  • Gestión priorizada de tareas
  • Asignación inteligente de empleados
  • Seguimiento de estado en tiempo real
  • Estadísticas y reportes

Menú del Sistema

El sistema presenta un menú interactivo en consola con las siguientes opciones:

1
Gestión de Tareas - Agregar, procesar, completar y eliminar tareas
2
Gestión de Empleados - Agregar, buscar y asignar empleados
3
Estadísticas - Ver reportes y métricas del sistema
4
Salir - Cerrar la aplicación

Consejo de Navegación: Utilice los números correspondientes a cada opción para navegar rápidamente por el menú. Presione Enter después de cada selección.

Gestión de Tareas

Agregar Nueva Tarea

1
Seleccione la opción "Agregar tarea" en el menú principal
2
Complete los datos solicitados:
  • ID único de la tarea
  • Descripción
  • Tipo (Urgente, Receta, Inventario, etc.)
  • Prioridad (Alta, Media, Baja)
  • Tiempo estimado en minutos
3
Confirme la creación de la tarea
[Ejemplo de formulario de tarea]

Formulario para agregar nueva tarea

Procesar Tareas

El sistema organiza automáticamente las tareas según su tipo y prioridad:

  • Tareas Urgentes: Van a la pila (LIFO) para procesamiento inmediato
  • Tareas Prioritarias: Se gestionan mediante cola de prioridad
  • Tareas Generales: Se almacenan en lista enlazada para procesamiento por orden

Gestión de Empleados

Registro de Empleados

Para agregar un nuevo empleado al sistema:

1
Seleccione "Agregar empleado" en el menú de gestión
2
Proporcione la información requerida:
  • ID del empleado
  • Nombre completo
  • Departamento
  • Puesto
  • Habilidades (separadas por comas)
3
El sistema asignará automáticamente una carga de trabajo inicial
[Formulario de empleado]

Formulario de registro de empleados

Asignación Inteligente de Tareas

El sistema utiliza un algoritmo de asignación óptima que considera:

  • Habilidades del empleado: Coincidencia con los requisitos de la tarea
  • Carga de trabajo actual: Distribución equitativa
  • Disponibilidad: Empleados con menor carga primero
  • Experiencia: Prioriza empleados con experiencia relevante
Consejo: Revise regularmente la carga de trabajo de los empleados para evitar saturación y mantener la productividad.
